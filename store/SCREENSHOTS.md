# App Store screenshot storyboard

Capture on a **physical iPhone** or Simulator at:
- **6.9"** — iPhone 16/17 Pro Max (required modern set)
- **6.5"** — iPhone 11 Pro Max / Xs Max (still accepted)

Tips: Light mode first for trust; add 1–2 dark frames. Hide Dynamic Island notch overlays in captions. Use real answers, not lorem.

| # | Frame | On-screen | Caption (under / on image) |

| 1 | Hero empty / first ask | Brand mark + “Ask anything. Privately.” + trust seals | **AI chat. No account. No dossier.** |

| 7 | iMessage bubble | Messages with Private AI answer card | **Ask in group chats** |

## Capture checklist
1. Fresh install or reset onboarding once for hero shot.
2. Seed 2–3 memory facts for the memory screenshot.
3. Use a clean sample Q: “Help me write a careful apology email” → polished answer.
4. Export PNGs lossless; don’t compress to mush.
5. In ASC, attach the same story to both size sets.

## Caption style
- Short, benefit-first, no jargon stack.
- Never say “we never see anything” stronger than PRIVACY.md — reviewers check.
- Prefer: *confidential compute*, *no account*, *on-device memory*.
